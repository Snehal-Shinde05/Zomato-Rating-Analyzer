📊 Zomato Rating Analyzer

1. Project Description
The Zomato Rating Analyzer is a data analysis project developed using Python and Jupyter Notebook.  
This project focuses on exploring and analyzing restaurant data from Zomato to understand patterns in ratings, pricing, and customer preferences.

The goal is to extract meaningful insights using data cleaning, visualization, and exploratory data analysis (EDA) techniques.

-

2. Objectives
- Analyze restaurant ratings distribution  
- Identify factors affecting ratings  
- Study pricing trends  
- Visualize insights using graphs  

---

3. Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook  

---

4. Dataset
- Zomato dataset (included in project)

---

5. Project Workflow
1. Data Collection  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  
5. Insight Generation
